A dark-themed personal finance tracker built with **Flutter**. This app allows users to manage multiple accounts, track transactions, and perform weekly financial calculations with persistent local storage.

## âœ¨ Features

- **Multi-Account Management:** Create and manage different wallets (Bank, Cash, Savings) with custom colors.
- **Transaction Tracking:** Log income and expenses with categories and timestamps.
- **Weekly Calculator:** A dedicated scratchpad to calculate 7-day earnings and spendings, featuring:
  - Persistent storage (your data stays even if you close the app).
  - Real-time "Savings" calculation.
  - One-tap data reset.
- **Dark UI:** Designed with a premium dark theme using **Google Fonts (Poppins)** and custom color palettes.
- **Local Persistence:** Powered by **SQLite** for fast, offline-first data management.
